# Ejner Hessel RSS viser

## Opgave A

Der skal laves en RSS viser i .net core eller mvc:
 - Lav en visning af RSS nyhedsitems, hvor man for hver item mindst kan læse: title og Publication date
 - Link til hver nyhedsitem så slutbrugeren kan navigere til nyhedsitem’et. 
 - Gør RSS URL konfigurerbar, således at man som slutbruger kan indsætte en valgfri RSS 
URL.
 - Giv brugeren mulighed for at angive hvordan listen skal sorteres.

## Krav

 - [ ] En side der kan læse RSS
 - [ ] Titlen og publication date
 - [ ] Link til hver nyhedsitems
 - [ ] Gør RSS Url konfigurerbar
 - [ ] Sorterings af RSS feed
